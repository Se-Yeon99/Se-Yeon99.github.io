## 강의
강의에서 배운 내용들
* random background - 랜덤으로 배경이미지 배치
* clock
* login - localStorage를 사용해 만든 로그인
* quotes - 새로 고침 시 랜덤으로 명언 출력
* todo - 추가와 삭제가 가능한 todo list
* weather - API를 이용해 위치 추적 후 날씨 출력
